# OpenClaw 免费 Token 导航

纯静态站点，无构建依赖，适合直接部署到 Vercel 或 GitHub Pages。

## 本地预览

```bash
python3 -m http.server 4173
```

然后打开 `http://127.0.0.1:4173`。

## 部署

### Vercel

- Import 这个仓库
- Framework Preset 选 `Other`
- `Build Command` 留空
- `Output Directory` 留空

### GitHub Pages

- 推送到 GitHub
- 在仓库 `Settings -> Pages`
- Source 选 `Deploy from a branch`
- Branch 选 `main`
- Folder 选 `/ (root)`
